- Displaying Data

```javascript
let cities = [
  { name: "Tokyo", population: 13960000 },
  { name: "London", population: 8982000 },
  { name: "Tahoe City", population: 2147 }
];

ReactDOM.render(
  <ul>
    {cities.map((city) => (
      <li>
        {city.name} | {city.population} people
      </li>
    ))}
  </ul>,
  document.getElementById("root")
);
```
